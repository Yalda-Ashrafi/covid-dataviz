# COVID-19 Data Visualization

This project visualizes COVID-19 case growth across countries using Plotly.

## Tools Used
- Python
- pandas
- Plotly

## Description
A line chart is used to compare total COVID-19 cases over time for USA, India, and Malaysia.

## How to Run
Install required libraries:
pip install pandas plotly

Run the script:
python covid_dataviz.py
